# Cassette Animation Project

## Overview
A simple animated cassette tape visualization using SVG and CSS animations. The project displays a retro cassette tape with rotating reels.

## Project Structure
- `index.html` - Main HTML file with SVG cassette tape graphic
- `styles.css` - Styling and animations for the cassette
- `server.js` - Express server to serve static files on port 5000
- `package.json` - Node.js dependencies

## Technology Stack
- HTML5
- CSS3 (animations)
- SVG graphics
- Express.js (static file server)

## Recent Changes
- 2025-11-03: Initial setup in Replit environment
  - Added Express server to serve static files on port 5000
  - Created package.json for dependency management
  - Set up workflow for development server

## Architecture
This is a static frontend application with an SVG-based cassette tape animation. The animation uses CSS keyframes to rotate the tape reels continuously.
